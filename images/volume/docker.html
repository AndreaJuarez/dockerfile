1.2 Introducción a Docker


¿QUÉ ES DOCKER COMMIT?

Docker es, una plataforma de software que nos permite crear, probar e implementar aplicaciones rápidamente. Docker empaqueta software en unidades estandarizadas llamadas contenedores que incluyen todo lo necesario para que el software se ejecute, incluidas bibliotecas, herramientas de sistema, código y tiempo de ejecución.

El comando docker commit de la plataforma de Docker nos ayudará a crear una imagen nueva a partir de un contenedor. Además, nos puede resultar útil confirmar los cambios o la configuración del archivo de un contenedor en una nueva imagen. Nos permite depurar un contenedor ejecutando un shell interactivo o exportar un conjunto de datos de trabajo a otro servidor. Sin embargo, es mejor usar Dockerfiles para administrar nuestras imágenes de una manera documentada y mantenible.
